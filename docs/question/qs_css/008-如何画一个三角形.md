# CSS 如何画一个三角形？

## 前言

在我们前端开发中，三角形也是我们遇到比较多的图案，通常使用图片或者 svg 完成效果图，因此今天我们使用纯 css 实现它。

## 实现

首先画一个矩形，然后给它添加四条边框，如下：
```html
<div class="box"></div>
```
```scss
.box {
  width: 100px;
  height: 100px;
  border-style: solid;
  border-width: 2px;
  border-top-color: #0ff;
  border-right-color: #000;
  border-bottom-color: #639;
  border-left-color: #f00;
}
```
